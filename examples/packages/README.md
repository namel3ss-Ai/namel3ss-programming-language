# namel3ss Package Examples

This directory contains real-world examples of namel3ss packages and workspaces demonstrating best practices for the module and package system.

## Examples Overview

### 1. Basic Package (`simple_package/`)
- Single package with multiple modules
- Basic dependency management
- Module exports and imports

### 2. Customer Service System (`customer_service_system/`)
- Multi-package workspace
- Complex dependency relationships
- Real-world AI application structure

### 3. Analytics Platform (`analytics_platform/`)
- Large-scale package organization
- Hierarchical module structure
- External package integrations

### 4. Microservices Architecture (`microservices/`)
- Service-oriented package design
- Inter-service communication patterns
- Scalable workspace organization

## Getting Started

Each example directory contains:

- `README.md` - Example description and setup instructions
- `namel3ss.toml` - Workspace or package configuration
- `*.ai` - namel3ss module files
- `docs/` - Additional documentation (where applicable)

To explore an example:

```bash
cd examples/packages/<example_name>
namel3ss packages list
namel3ss modules list
```
