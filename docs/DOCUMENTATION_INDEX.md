## üÜò Quick Reference

### Syntax Essentials

- **Indentation**: Use 4 spaces (not 2 or tabs)
- **Memory blocks**: `memory "name": { scope: "session", kind: "key_value" }`
- **No ternary operators**: Use `if/elif/else` blocks
- **Page structure**: Consistent 4-space indentation after `page "Name" at "/":` 

### Common Patterns

```n3
# Model configuration
model chat_model:
    provider: openai
    name: "gpt-4"

# Memory store
memory "history":
    scope: conversation
    kind: conversation

# AI chain
chain "process":
    inputs:
        text: text
    steps:
        - step generate:
            model: chat_model
            output: response

# Page with form
page "Home" at "/":
    show form "MyForm":
        field "input" type text
        button "Submit"
        on submit:
            show toast "Done!"
```
